# HACKATHON IA FOR GOOD

## Prérequis

- nodejs
- npm

## Installation

- Cloner le repo

```bash
git clone https://github.com/yulannn/ia.git
```

```bash
cd client
```

- Installer les dépendances

```bash
npm install
```

## Lancer le projet

```bash
npm run dev
```

- Une fois le projet lancé, copier-coller l'adresse ci-dessous dans votre navigateur :
```bash
http://localhost:5173/
```
